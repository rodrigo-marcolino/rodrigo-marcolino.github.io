Problem Solving

By Rodrigo Marcolino | Posted on: 15 July 2021

Let's start with a definition:

Problem solving refers to cognitive processing directed at achieving a goal when the problem solver does not initially know a solution method. A problem exists when someone has a goal but does not know how to achieve it. Problems can be classified as routine or nonroutine, and as well defined or ill defined. The major cognitive processes in problem solving are representing, planning, executing, and monitoring.

Richard E. Mayer The Oxford Handbook of Cognitive Psychology

Definitely Learning how to define a problem, break it into manageable parts, and work toward a solution it is a core skill to become a Software Developer. We will find in almost all job descriptions for developers the skill "problem solving".

Therefore, we could reach the conclusion that we must have this skill in order to be a developer. Probably you are asking you self, "Do I have this skill? Can I learn it?".

Steps for Problem Solving

1. Read the problem as many times as necessary to obtain a clear understanding of the problem

2. Work through the problem with sets of sample data

3. Simplify and optimize your steps

4. Write pseudocode

5. Translate pseudocode into code and debug

6. Simplify and optimize your code
